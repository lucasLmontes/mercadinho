Sobre o site

Este é um projeto individual em Angular voltado para o aprendizado em programação front-end.

Por que criar este projeto?

Pela variedade de mecanismos diferentes que podem ser implementados no contexto de e-commerce, não se limitando somente à exposição de produtos, mas também a outras possibilidades mais abrangentes como, validação de usuário, adição de itens em um elemnto específico (carrinho de compras), validações via e-mail etc.

Acesse o git para ter acesso ao código do projeto
